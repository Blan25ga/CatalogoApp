## 📦 CatalogoApp 📦

Aplicación de escritorio desarrollada en **C# .NET Framework (WinForms)** para la gestión de artículos, marcas y categorías.  
El proyecto está orientado a prácticas de CRUD y conexión con bases de datos SQL Server.

---

## 🚀 Características principales
- Interfaz gráfica con formularios (`Form1`, `frmAltaArticulo`).
- Operaciones CRUD sobre artículos.
- Gestión de marcas y categorías.
- Arquitectura organizada en capas:
  - **Clases** → Modelos de datos.
  - **Servicios** → Lógica de negocio y acceso a datos.

---

## 🛠️ Requisitos
- **Visual Studio 2022** (o superior).
- **.NET Framework 4.8.1**.
- **SQL Server** (local o remoto).
- Conexión configurada en `App.config`.
